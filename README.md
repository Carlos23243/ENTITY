# ENTITY
# Modelado de Lógica Combinacional en VHDL
El objetivo de este proyecto es demostrar el uso de VHDL para describir el comportamiento y la estructura de circuitos digitales básicos. Se aplican conceptos fundamentales como la separación de ENTITY y ARCHITECTURE, y el uso de sentencias concurrentes.
# Componentes Incluidos:
Multiplexor 2 a 1: Selección de señales mediante una entrada de control.

Decodificador 2 a 4: Activación de salidas específicas basadas en una dirección de entrada.

Sumador Completo: Implementación de aritmética binaria con manejo de acarreo (carry).

Sumador de 4 Bits: Diseño jerárquico/comportamental para suma de vectores de datos.
